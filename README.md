## Chapter02 - 시간 복잡도
| 알고리즘 | 시간 복잡도    |
|------|-----------|
| 이진 탐색 | O(logN)   |
| 선형 탐색 | O(N)      |
| 정렬   | O(N logN) |
| 조합   | O(2^n)    |
| 순열   | O(N!)     |

시간 복잡도를 계산할 땐 O(N), O(N+M)에 문제 제한사항에 명시된 가장 큰 입력을 대입하여 효율적인 코드인지 확인
